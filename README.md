# arduino-dc-motor
## Control a DC Motor with an Arduino

### Hardware required
- 1 x L298n module (or L293d H-bridge Motor Driver)
- 1 x DC motor
- 1 x Arduino uno r3
- jumper wires (a bunch of them)
- Power source (eveready battery LOL)
